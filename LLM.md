## ELI10 TLDR
This project builds a 3D rotating ring of images that spins as you scroll. It uses "Framer Motion" to make the movement smooth and "CSS 3D" to make it look like a real object in space. No Tailwind is used; everything is styled with clean JavaScript objects.
